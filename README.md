# ncbi-blast-legacy

## Introduction

This is the legacy blast software.

## Installation

Clone this repository, and:

```bash
mkdir -p include bin lib build altlib shlib
cd build
../make/makedis.csh
```

If compilation fails on the first try, run the script again.

This version supports only Linux x86_64.
